# 更新日志

所有本项目的重要变化都会记录在此文件中。

## [1.0.0] - 2026-05-22

### ✨ Added（新增）
- 初始版本发布
- 局域网文件传输核心功能
- 拖拽文件/文件夹上传
- 二维码连接生成
- 连接码快速配对
- 传输文件列表显示
- 已接收文件列表
- 完整的传输历史记录
- 设备列表展示（显示连接设备 IP）
- 实时日志输出

### 🎨 Features（功能特性）
- 现代化 WPF 用户界面
- 拖拽操作支持
- 多设备同时连接
- 实时状态显示
- 文件管理和共享
- 连接信息一键复制

### 🛠️ Technical（技术）
- 基于 .NET 8.0 框架
- WPF 前端框架
- MVVM 架构设计
- QRCoder 二维码生成
- CommunityToolkit.Mvvm 依赖注入

## 更新说明

### 版本号规则

采用 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **MAJOR** (主版本号) - 重大功能更新，可能不兼容之前版本
- **MINOR** (次版本号) - 新增功能，向下兼容
- **PATCH** (修订版本号) - 问题修复，向下兼容

### 标签说明

- ✨ **Added** - 新增功能
- 🎨 **Changed** - 功能改动
- 🐛 **Fixed** - 问题修复
- 🔒 **Security** - 安全相关
- ⚡ **Performance** - 性能优化
- 📝 **Deprecated** - 废弃功能
